🗂️ Almoxarifado — Sistema de Controle de Estoque

Sistema web de controle de estoque desenvolvido para o setor de Enfermagem, permitindo o cadastro e a listagem de materiais com integração a uma API na nuvem.

🚧 Status do Projeto


Em desenvolvimento — Este projeto está sendo construído em sprints. Atualmente a Sprint 1 foi concluída. As Sprints 2 e 3 ainda serão implementadas.

✅ O que foi feito até agora (Sprint 1)


Estrutura HTML com os elementos obrigatórios do contrato técnico
Formulário de cadastro de materiais com nome e quantidade
Tabela de listagem de materiais
Integração com MockAPI via requisição GET ao carregar a página
Integração com MockAPI via requisição POST ao cadastrar um novo material
Estilização responsiva para desktop e mobile
Mensagens de feedback para o usuário



🛠️ Tecnologias utilizadas


HTML5
CSS3
JavaScript (Vanilla)
MockAPI — API REST simulada na nuvem



📋 Contrato Técnico

Os seguintes IDs estão implementados conforme exigido pelo autograding:


input-nome — Input do nome do material
input-quantidade — Input da quantidade
btn-cadastrar — Botão de cadastro
lista-materiais — Tabela de listage
